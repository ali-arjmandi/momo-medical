# Documentation

This folder contains the diagrams for the Nurse Call System assignment.

## Diagrams

- **Class Diagram** (`class-diagram.*`) - Domain model showing relationships between entities
- **Sequence Diagram** (`sequence-diagram.*`) - Application flow showing interactions between components

## File Formats

Diagrams are provided in both source format (for editing) and rendered format (for viewing):
- `.puml` - PlantUML source files
- `.png` - Rendered images
- `.mmd` - Mermaid source files (alternative)
